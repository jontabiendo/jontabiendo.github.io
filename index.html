Projects

EpicEats:

EpicEats is an UberEats clone group project with additional inspiration taken from DoorDash. Logged-in users can browse restaurants and their menus, create and edit orders, finalize orders, leave reviews and view past orders Technologies used to build this project include javascript, react, redux, css, aws, python and Postgresql.

Futures features my team and I are working on implementing include:

  • Maps - Implementing the Google Maps API to search and filter by real-world location

levelUp:

levelUp was done as my capstone solo project. It is a retro-style to-do list and task-management web application for teams and/or individuals. As a single-page application it was particularly challenging to make the entire page responsive to any new data given. Whenever a change is saved and committed, the entire page will reflect these changes without needing a refresh. Technologies used to build this project include javascript, react, redux, css, python and Postgresql.

Future features include:

  • Notes/comments - Members will be able to add notes to tasks to ask questions and give updates on that task
  • Friends/connections - Users will be able to send and accept friend requests to see their friend's public lists
  • Messaging - Friends and team members will be able to directly message each other or the group

airbee'n'bee:

airbee'n'bee is an Airbnb clone website where users can log-in, sign-up, browse spots, leave reviews, as well as create and manage spots. Technologies used to build this project include javascript, react, redux, css, aws and Postgresql. I'm continuously learning and practicing to strengthen my software development skills.

Future features include but aren't limited to:

  • Booking reservations - Creating new, editing existing, and cancelling resrvations for users and owners
  • Maps - Implementing the Google Maps API to search and filter by real-world location
  • Messaging - Allow users to contact spot owners and owners to contact users with reservations to their spots

Games

This section is for some of the games I've contributed to as a developer.

9 Door Dungeon:

9 Door Dungeon is a 2D platform fighting game I created as a solo-developer for the 9 day game jam hosted by Buffalo Game Space. As a solo-developer I had the opportunity to have complete ownership over the game and bore all the responsibility of designing, planning and building the game. I used Phaser, a JavaScript game engine for the browser, for the first time and gained a lot of knowledge and practice researching new docs and issues/solutions other developers faced when facing similar obstacles.

2D game assets were acquired from CraftPix.com, backgrounds and splash art was made with Leonardo.ai, and music was acquired from EpidemicSound.com.

CuckooClimb:

CuckooClimb is a 2D platform game that I created with a small team of developers during the 19th Bufallo Game Jam, a 48-hour event where different teams created video games from scratch. We built CuckooClimb with Unity in C# to integrate the assets our artists created with our game environment and mechanics.
